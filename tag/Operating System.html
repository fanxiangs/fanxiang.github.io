#Operating System

What is CPU？

本文介绍了 CPU 的定义和组成，以及 CPU 的发展历程和性能指标。其中，Clock Speed、Cores、Cache 和 Hyper-Threading 是影响 CPU 性能的关键指标。此外，还介绍了如何检查 CPU 的关键指标，如 CPU 使用率、CPU 负载平均值、CPU 时钟频率、CPU 核心数量、CPU 线程数量和 CPU Scaling Governor等。
